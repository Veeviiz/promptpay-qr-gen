# PromptPay QR Generator

โปรเจคนี้เป็นแอป React + TypeScript ที่ใช้ Vite สำหรับพัฒนาและ build.

## วิธีติดตั้ง

เปิด terminal ในโฟลเดอร์โปรเจค แล้วรัน:

```bash
npm install
```

## วิธีรันในโหมดพัฒนา

รันคำสั่งนี้เพื่อเริ่มเซิร์ฟเวอร์พัฒนา:

```bash
npm run dev
```

จากนั้นเปิด URL ที่แสดงใน terminal (ปกติ `http://localhost:5173`).
